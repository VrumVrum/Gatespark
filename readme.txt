== Installation ==

1. Upload the plugin files to `/wp-content/plugins/gatespark-revolut` or install directly from the WordPress Plugins screen.
2. Activate the plugin through the ‚ÄòPlugins‚Äô menu in WordPress.
3. Go to **WooCommerce ‚Üí Settings ‚Üí Payments ‚Üí GateSpark**.
4. Add your Revolut API key (Sandbox or Live).
5. Copy the generated webhook URL and add it in your Revolut Business Dashboard.
6. Save settings and start accepting payments.

== Frequently Asked Questions ==

= Do I need a Revolut Business account? =
Yes. You need a Revolut Business account with API access.

= Does it support Sandbox and Live modes? =
Yes, you can switch between Sandbox and Live directly in the settings.

= Is GateSpark compatible with HPOS? =
Yes. It declares compatibility with WooCommerce High Performance Order Storage.

= Does it work with Subscriptions? =
Not yet. Subscriptions and saved cards are planned for future releases.
